# Video and Music Downloader on YouTube

## Build

```shell
cargo build --release
```

## Usage

### mp4

```shell
./target/release/yt-downloader -u https://www.youtube.com/watch?v=XWrvbDkmatw -f mp4
```

### mp3

```shell
./target/release/yt-downloader -u https://www.youtube.com/watch?v=XWrvbDkmatw -f mp3
```
